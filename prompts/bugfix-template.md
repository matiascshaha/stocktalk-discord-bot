# Bugfix Prompt Template

Bug summary:
<Observed behavior>

Expected behavior:
<Expected outcome>

Reproduction:
<Steps, env, data>

Required process:
1. Identify likely root causes with evidence (`path:line`).
2. Propose smallest safe fix.
3. Add or update tests reproducing the bug.
4. Run targeted tests, then broader validations.
5. Provide before/after behavior summary.

Output format:
- Root cause
- Patch summary
- Tests executed and results
- Residual risk
